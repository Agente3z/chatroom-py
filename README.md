# ChatRoom-py

**Simple chat room written in python using flask as to create a server and requests**

## Requirements

**SERVER:**
- Flask
- Flask_SocketIO

  $ python -m pip install flask flask_socketio

**CLIENT:**
- Requests
- Python-SocketIO
- Pillow
- Numpy

  $ python -m pip install requests python-socketio pillow numpy
